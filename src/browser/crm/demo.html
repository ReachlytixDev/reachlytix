Architecture Overview

The CRM is designed as a modular system that integrates seamlessly with VS Code's existing architecture:

  • Component-based UI with reusable shadcn/ui elements
  • TypeScript interfaces for type safety and better DX
  • Tailwind CSS with custom prefix to avoid conflicts
  • Patch-based integration with VS Code's workbench
  • Service-oriented architecture for extensibility
